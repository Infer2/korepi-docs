## เปลี่ยนมอนสเตอร์:
- เมื่อเปิดใช้งาน, มอนสเตอร์ที่เรนเดอร์ใหม่ทั้งหมดจะถูกเปลี่ยนเป็นมอนสเตอร์ที่เลือก
### ยกเว้นสัตว์:
- เปิดหรือปิดการเปลี่ยนสัตว์
### ID มอนสเตอร์ที่กำหนดเอง:
- ใช้ ID แทนรายการที่ให้ไว้
### สุ่มมอนสเตอร์:
- การเปลี่ยนแปลงจะเป็นการสุ่ม
